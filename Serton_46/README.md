# סרטון 46

## Content

תמונה של סבתא משכנעת את סבא אני לא יכולתי לעבוד, אני הייתי ילד מאוד עדין, ולא יכולתי לעבוד בעבודות האלה, לא היה לי כוח. אמא שלי דיברה עם אבא שלי, והיא אמרה לו: אנחנו מוכרחים לצאת מהמושב הזה. הוא לא בשביל זה, אתה לא רואה, אתה בסדר, אבל אנחנו צריכים להמשיך, והוא לא יכול לעזור לך, הוא איש מקצוע, אתה לא רואה איזה ראש מקצועי יש  לו?

## Context / Setup
