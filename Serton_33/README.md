# סרטון 33

## Content

לקחנו רכבת מבאד רייכנהאל גם-כן עם איזה מדריך. ברכבת לא היו בכלל חלונות אחרי המלחמה, שוב מפה ונסענו לכיוון קאסל (Kassel). תמונה של תחנת רכבת גדולה כשבאנו לקאסל יורדים בתחנה, זאת תחנת רכבת גדולה מאוד. כשיצאנו החוצה לא ראינו שום דבר, הכל היה מגולח רק התחנה נשארה, האמריקאים פוצצו את הכל מסביב, נשארה רק התחנה. משם באו משאיות.

## Context / Setup

כמה משאיות של יהודים העבירו אותם עד הגשר, בגשר עברנו, אמרו שזה חורף היה כבר חורף, היינו שומעים את המים זורמים מההרים, ועברנו. לא היה לנו כסף, אז אבי שם סוליה אחת מגומי וסוליה אחת מעור, אז הוא החליק למטה, והגענו לבאד רייכנהאל (Bad Reichenhall), באד רייכנהאל זה איפה שקן הנשרים של היטלר. היינו שם בבאד רייכנהאל, היה שם באמת אוכל טוב.
