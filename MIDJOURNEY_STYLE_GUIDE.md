# Midjourney Style Guide - Tasrit Documentary

## Universal Style Parameters

**APPLY TO ALL IMAGE AND VIDEO PROMPTS:**

### Visual Style
```
Style: 2D animation, moving painting, animated historical illustration
Tone: Serious, evocative, melancholic, threatening
Medium: Painted animation style
```

### Color Palette
```
Heavily desaturated, almost monochromatic
Sepia tones, muted browns, deep greys
Faded memory aesthetic
Grim atmosphere
```

### Lighting
```
High-contrast chiaroscuro
Long oppressive shadows
Dramatic light and shadow interplay
Melancholic lighting
```

### Character Design
```
Stylized but expressive
Emotion conveyed through posture and expression
Not realistic detail, but powerful emotion
Historical 1940s European features
```

### Historical Setting
```
Period: 1930s-1940s Romania and Europe
Authentic architecture, clothing, objects
War-era atmosphere
```

## Camera & Animation Vocabulary

### Camera Movements
- **Pan** - Horizontal camera movement
- **Tilt** - Vertical camera movement
- **Zoom in** - Moving closer to subject
- **Zoom out** - Revealing more of scene
- **Handheld** - Subtle shake, organic feel
- **Static** - No camera movement, only subject movement
- **Tracking** - Following subject movement
- **Crane** - Elevated sweeping movement

### Atmospheric Elements
- **Dust particles** - Floating in light beams
- **Smoke** - From trains, buildings
- **Flickering light** - Candles, fire
- **Wind** - Moving fabric, papers, leaves
- **Rain** - Streaking windows, puddles
- **Snow** - Falling, settling
- **Fog/mist** - Obscuring distance

### Emotional Pacing
- **Slow, contemplative** - Melancholy, reflection
- **Urgent, tense** - Fear, escape
- **Static, oppressive** - Dread, waiting
- **Gentle, tender** - Memory, family

## Prompt Structure Template

### Image Prompt Template:
```
[Setting description], [composition and framing], [character(s) and action],
[lighting description], [emotional atmosphere], 2D animated historical illustration,
heavily desaturated sepia tones, muted browns and greys, high-contrast chiaroscuro,
long oppressive shadows, serious evocative tone, 1940s [location], moving painting style
```

### Video Prompt Template:
```
[Camera movement], [atmospheric elements], [character movement],
[light behavior], [emotional pacing], subtle [specific details],
maintaining melancholic threatening atmosphere
```

**Image Prompt:**

## Quality Standards

- **Specificity:** Every prompt must clearly describe what to show
- **Consistency:** Maintain style across all 48 segments
- **Emotion:** Every shot must convey clear emotional intent
- **Period Accuracy:** Authentic 1940s details
- **Cinematic:** Think in film language
- **Brevity:** Concise but complete

## Location-Specific Elements

### Romania (Bucharest, Ia»ôi)
- Cobblestone streets
- Multi-story buildings with balconies
- Tram lines overhead
- 1930s-40s European architecture
- Orthodox church domes in background

### DP Camps (Austria, Germany)
- Barracks/temporary structures
- Barbed wire fences
- Muddy grounds
- Military-style buildings
- Crowded conditions

### Journey Scenes
- Period trains (steam)
- Rural landscapes
- Border crossings
- Night scenes with minimal light
