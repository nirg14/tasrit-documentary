## ğŸ¬ Production Workflow

### For Each Serton:

1. **Open SHOTS.md file** in the Serton folder
2. **Review scene overview** (setting, tone, action)
3. **For each shot:**
   - Read the shot description
   - Copy **Image Prompt** into Midjourney
   - Generate first frame
   - Download and save to `images/` folder
   - Copy **Video Prompt** into Midjourney video
   - Animate the generated frame
   - Download animated shot

4. **Sequence shots** in editing software
5. **Add narration** from `script.txt`
6. **Check off** items in `production_notes.md`
