# סרטון 36

## Content

עד שיום אחד בבוקר אנחנו קמים, ומוקפים עם צבא אמריקאי. מה קרה? הם החליפו גנרל. היה שם גנרל אחד בשם קליי, הייתי תולה אותו על עץ הכי גבוה למרות שהוא היה גנרל אמריקאי. תמונה של גנרל אמריקאי עושה חיפוש הוא נכנס לתוך המחנה עם הצבא, התחיל לחפש יהלומים, התחיל לחפש דולרים, התחיל לחפש כל מיני רכוש. הם באו היו ג'נטלמנים, לבושים כמו בויטנאם שהולכים לקרב עם טנקים, כי הם חשבו שיהיה מרד או משהו כזה, ולא יתנו להם להיכנס.

## Context / Setup

אבי עבד באונרייא בתור חייט, היתה לו משכורת, היה שם אוכל טוב, קנינו כמה דברים כמה שיכולנו. תמונה של הכרזת המדינה  1-3 ברקע השמעה של הכרזת המדינה של בן גוריון בחמישה-עשר למאי 48' היתה הכרזת המדינה, אז אני זוכר ששמענו ברמקול בחוץ כל היהודים בשעה חמש היו חדשות, ואמרו לנו שכבר הקימו את המדינה, שמענו את הקולות. התחילו להיות תהלוכות ושמחות בתוך המחנה וזהו.
